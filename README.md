# 👨‍💼 EmployEdge - Employee Management System

A modern and responsive **Employee Management System** designed to simplify employee record management. EmployEdge enables organizations to efficiently manage employee information through an intuitive interface while demonstrating core **Database Management System (DBMS)** concepts.

🌐 **Live Demo**  
https://dbms-team12-a1-kegiyvifz-sirireddy1910s-projects.vercel.app/

---

## 📖 About the Project

EmployEdge is a web-based Employee Management System that allows users to manage employee information efficiently. The project demonstrates the implementation of CRUD operations, database design principles, and responsive web development.

This project was developed as part of our **Database Management Systems (DBMS)** coursework to apply database concepts in a real-world application.

---

## ✨ Features

- 👨‍💼 Employee Record Management
- ➕ Add New Employees
- ✏️ Update Employee Information
- 🗑️ Delete Employee Records
- 🔍 Search Employee Details
- 🏢 Department Management
